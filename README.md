# HyperZoneLogin

实现多种登入形式的API框架兼少数标准实现

## 兼容情况

### 可导入数据的插件

- [x] AuthME
- [x] MultiLogin

### 兼容性
登入方式：支持主流登入方式  
MC版本：1.19.1+暂无问题 低于1.19.1则只能使用域名作为识别方式  
启动器：支持市面常见
